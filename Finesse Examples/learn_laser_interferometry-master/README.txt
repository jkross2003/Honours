Learn Laser Interferometry
---------------------

This is a self studie course and reference material for simulating laser interferometry.

If you do not have IPython/Jupyter Notebook, FINESSE, and PyKat installed, read the
installation instructions in Getting_started.pdf.

Otherwise, start IPython/Jupyter notebook in the same folder you found this file
and start notebooking! Have fun!
